Calc in terminal

My former boss, Luk치코 Me코콘an, created "Simple bash calculator" which adds you new command =. Now, you no more need to open calculator when you just want to do some easy ( or even more complex ) math.

Usage

# addition
root@pc:~$ = 5+3                                                
8

# division
root@pc:~$ = 4/2
2.00000000000000000000

# square root 
root@pc:~$ = "scale=30; sqrt(2)"
1.414213562373095048801688724209

# power
root@pc:~$ = "6^6"
4665

# convert from binary to decimal 
root@pc:~$ = "ibase=2; obase=A;1100"
12

# arctangent
root@pc:~$ = "a(1)"
.78539816339744830961
