Každý programátor má nejaké svoje obľúbené tooly / nástroje, ktoré mu pomáhajú pri práci. Zopár takýchto nástrojov, ktoré sa používajú pri vývoji webových stránok by som rád spomenul s posunul ďalej.

Every programmer has some favorite tools which helps him during work. I want to mention a few of these tools using when developing web pages.

Online tools list

JS Bin

If you need to quickly try something, then jsbin may be an option. It is immediately showing changes as you are typing so you can see if everything works as you wish or not.

jsbin.com

JSHint

jshint.com

Can I use

Jednoduchá a prehľadná stránka, kde sa dá veľmi rýchlo nájsť, ktorý prehliadač podporuje danú technólogiu a od ktorej verzie. Stačí napríklad zadať slovo "picture" a z výpisu je hneď jasné, že pri Safari a Internet Exploreri nastane problém.

caniuse.com

DevDocs

Documentation and examples for JavaScript functions, CSS styles, DOM elements, events, HTTP state codes, etc...

devdocs.io

Code Beautify

Are you working with JSON or XML? Do you need to minify JSON? Or contrariwise you have minimized version and you want it to be nicely formatted? Maybe you want to change the format from JSON to CSV? There are many options and new features are being still added.

codebeautify.org

JSON parser online

Similar functionality as Code Beautify, but in this case, changes are reflected immediately while editing the input along with its validation. The output can be formatted or minified with the option of type and index highlighting.

json.parser.online.fr/beta

GTmetrix

Analyze page and then generates report, where it evaluates optimization of the page. E.g.: if it uses minimized JavaScript / CSS, which images to merge into one sprite + page load time, size and a lot other stuff.

gtmetrix.com

WebPagetest

Niečo podobné ako GTmetrix, len graficky inak spracované, možno to bude niekomu viac sedieť.

Something like GTmetrix, but

www.webpagetest.org

Facebook debugger

Chceš zistiť, ako bude vyzerať tvoja stránka pri zdielaní na facebooku? V debuggeri zobrazuje náhladový obrázok, popis a ďalšie informácie, ktoré dokázal vyčítať zo stránky. Tiež zobrazí prípadné chyby alebo problémy, ktoré nastali pri čítaní.

developers.facebook.com/tools/debug

Chrome extensions

ColorZilla

Sem tam sa stane, že potrebujem rýchlo zistiť kód nejakej farby a toto je ideálny spôsob. Prípadne, ak treba vygenerovať v CSS rôzne prechody medzi farbami.

www.colorzilla.com/chrome

Meta Inspector

Dostal si zo SEO za úlohu doplniť do stránky pekné popisy ( meta description ), open graph tagy pre facebook a iné magické vecičky a pri kontrole sa ti to nechce stále hľadať v zdrojovom kóde? Síce to nie je najkrajšie graficky spravené, ale účel si to plní.

www.omiod.com/meta-inspector.php

Niečo málo k databázam

phpMyAdmin

www.phpmyadmin.net

pgAdmin

www.pgadmin.org

Elastic Sense

www.elastic.co/blog/sense-2-0-0-beta1

Conclusion

There are a lot of such tools on the internet and everybody is using something else. When I find some other worth-to-mention tool or you are using one I did not write about and you want to recommend it, write me and I will add it ;-)
