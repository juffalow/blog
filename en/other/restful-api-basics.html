Methods

GET

This method is used to read / retrieve resource or a collection of resources from the server. This makes GET safe operation. If the request was successful, status 200 is returned, or if the specific resource doesn't exist, 404 is returned.

POST

POST is used to create new resources. If there is no ID, the server assigns it an internal ID, otherwise the specified ID is used. It returns status 201 on successful creation, 409 if resource already exists.
