Factorial

This is a tricky one. Your mission is to write a SELECT which output is a factorial of given number. When I asked this my colleagues or friends, they were like to write a procedure to solve this. But it has to be just SELECT! If you do not know any other database system then MySQL, you can find this impossible.

The solution lies in recursive query. Not every database system has it, but for example in Oracle or Postgre there is keyword WITH.

WITH factorial AS (
    SELECT 1 AS fac, 1 AS num
    UNION ALL
    SELECT fac * ( num + 1 ), num + 1
    FROM fact
    WHERE num < 12 )
SELECT fac
FROM factorial
WHERE num=5;

Group concatenation

Once my friend asked me how to concatenate string columns when group by function is used. It is easy : you should write a select that concatenate string column and separate the values with ( for example ) comma.

group_byvalue
1Monday
2Thursday
3Friday
3Sunday
1Wednesday
2Friday

When you write a basic group by select ( SELECT group_by, value FROM test_table GROUP BY group_by, value; ), the result will be something like this :

group_byvalue
1Monday
1Wednesday
2Thursday
2Friday
3Friday
3Sunday

So how to get the values in one row separated with comma? The solution is in group_concat function ( in MySQL! ). Than you are able to write SELECT group_by, group_concat(value) FROM test_table GROUP BY group_by;. This outputs :

group_byvalue
1Monday,Wednesday
2Thursday,Friday
3Friday,Sunday
