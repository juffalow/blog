Každý programátor má nejaké svoje obľúbené tooly / nástroje, ktoré mu pomáhajú pri práci. Zopár takýchto nástrojov, ktoré sa používajú pri vývoji webových stránok by som rád spomenul s posunul ďalej.

Zoznam online nástrojov

JS Bin

Ak potrebuješ len narýchlo niečo vyskúšať, tak jsbin môže byť jedna z možností. Ako píšeš, tak sa hneď zobrazujú zmeny a krásne vidíš, či všetko funguje tak ako chceš alebo nie.

jsbin.com

JSHint

jshint.com

Can I use

Jednoduchá a prehľadná stránka, kde sa dá veľmi rýchlo nájsť, ktorý prehliadač podporuje danú technólogiu a od ktorej verzie. Stačí napríklad zadať slovo "picture" a z výpisu je hneď jasné, že pri Safari a Internet Exploreri nastane problém.

caniuse.com

DevDocs

Dokumentácia a príklady k JavaScriptovým funkciám, CSS štýlom, DOM elementom, eventom, HTTP stavovým kódom, atď...

devdocs.io

Code Beautify

Pracuješ s JSON alebo XML? Potrebuješ minimalizovať JSON? Alebo naopak máš jeho minimalizovanú verziu a chceš ju mať pekne naformátovanú? Alebo chceš zmeniť formát z JSON na CSV? Možností je veľa a nové funkcie stále pribúdajú.

codebeautify.org

JSON parser online

Podobná funkcionalita ako pri Code Beautify, ale v tomto prípade sa zmeny prejavujú hneď pri edite vstupu spolu s jeho validáciou. Výstup môže byť formátovaný alebo minimalizovaný s možnosťou zvýraznenia typov a indexov.

json.parser.online.fr/beta

GTmetrix

Analyzuje stránku a následne vygeneruje report, kde zhodnotí optimalizáciu stránky. T.j.: či sa využíva minimalizovaný JavaScript / CSS, ktoré obrázky je dobré spojiť do spritu + doba načítania stránky, jej celková veľkosť a veľa iného.

gtmetrix.com

WebPagetest

Niečo podobné ako GTmetrix, len graficky inak spracované, možno to bude niekomu viac sedieť.

www.webpagetest.org

Facebook debugger

Chceš zistiť, ako bude vyzerať tvoja stránka pri zdielaní na facebooku? V debuggeri zobrazuje náhladový obrázok, popis a ďalšie informácie, ktoré dokázal vyčítať zo stránky. Tiež zobrazí prípadné chyby alebo problémy, ktoré nastali pri čítaní.

developers.facebook.com/tools/debug

Rozšírenia pre prehliadač Chrome

ColorZilla

Sem tam sa stane, že potrebujem rýchlo zistiť kód nejakej farby a toto je ideálny spôsob. Prípadne, ak treba vygenerovať v CSS rôzne prechody medzi farbami.

www.colorzilla.com/chrome

Meta Inspector

Dostal si zo SEO za úlohu doplniť do stránky pekné popisy ( meta description ), open graph tagy pre facebook a iné magické vecičky a pri kontrole sa ti to nechce stále hľadať v zdrojovom kóde? Síce to nie je najkrajšie graficky spravené, ale účel si to plní.

www.omiod.com/meta-inspector.php

Niečo málo k databázam

phpMyAdmin

www.phpmyadmin.net

pgAdmin

www.pgadmin.org

Elastic Sense

www.elastic.co/blog/sense-2-0-0-beta1

Záver

Takýchto toolov je istotne viac a každý má svoje obľúbené. Keď natrafím na nejaké ďalšie, ktoré by stáli za zmienku, alebo používaš taký, ktorý tu nemám a chcel by si ho odporučiť, napíš mi a ja ho sem pridám ;-)
